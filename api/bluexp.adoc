NetApp BlueXP  BlueXP  是一个统一控制平台、可提供一个混合多云平台、用于跨内部环境和公共云环境管理存储和数据服务。它由多个不同的服务或组件组成、每个服务或组件都公开一个关联的REST API。请参见下面提供的文档和相关资源、开始使用BlueXP  REST API实现BlueXP  环境自动化。

.BlueXP  REST API
您可以使用各种REST API自动管理由BlueXP  管理的存储和数据服务。
